## 詳しい説明（4000文字以内・プレーンテキスト）

> Play Console はMarkdownを描画しないため、以下のプレーンテキストをそのまま貼り付けること。

```
交通費マネージャーは、日々の交通費や交際費をかんたんに記録・管理できる個人向け支出管理アプリです。


■ こんな方におすすめ

・電車・バス・タクシーの交通費を毎日記録したい方
・カフェや食事など交際費の把握に使いたい方
・シンプルで使いやすい家計管理アプリを探している方
・プライバシーを重視し、データをクラウドに送りたくない方


■ 主な機能

【支出の記録】
日付・金額・カテゴリ・サブカテゴリ・メモを入力して支出を記録します。満足度（1〜5段階）を記録しておくと、あとから「コスパが良かった支出」を振り返るのに役立ちます。

【カテゴリ・サブカテゴリ管理】
「交通費」と「交際費」の2つのカテゴリで管理します。
交通費のサブカテゴリ: 電車 / バス / タクシー / 駐車場 / その他
交際費のサブカテゴリ: 食事 / カフェ / 飲み / ギフト / エンタメ / その他

【テンプレート機能】
よく使う支出パターンをテンプレートとして登録しておくと、次回から入力を素早く完了できます。テンプレートは並び替えや編集・削除も自由に行えます。

【頻出金額の候補表示】
直近の記録から頻繁に使われた金額を自動で候補表示します。タップするだけで金額入力が完了するため、毎日の記録がスムーズになります。

【サマリー・集計画面】
今月・先月・過去3ヶ月の期間で支出を集計できます。カテゴリ別の内訳・構成比、満足度の平均と分布を一覧で確認できます。

【CSVエクスポート】
記録したデータをCSVファイルとして書き出せます。ExcelやNumbersに取り込んで、さらに詳しく分析することができます。


■ プライバシー・データの安全性

入力したデータはすべてお使いのスマートフォン内に保存されます。インターネット接続は一切必要ありません。クラウドへのデータ送信や外部サービスとの連携は行いません。

CSVエクスポートも、あなたが「エクスポート」ボタンをタップした場合のみ、端末内にファイルを作成します。第三者にデータが渡ることはありません。


■ こんなシーンで活躍

・通勤の電車賃を毎日つけておきたい
・月末に交通費の合計を会社に申請したい
・ランチ・カフェ代の出費を月ごとに把握したい
・週末の遊び代をざっくり記録しておきたい


■ オフライン完全対応

Wi-Fi・モバイルデータ通信がない環境でも問題なく使えます。地下鉄の中、旅先のどこでも記録できます。


■ シンプルで直感的な操作

片手でも操作しやすいデザインを採用しています。支出の登録は最短3タップで完了します。前回の入力値を次の登録時に引き継ぐ機能もあり、毎日の記録が続けやすくなっています。
```
